# dotfiles
Meus arquivos dotfiles

# Instalação

```bash
curl -sSL https://raw.githubusercontent.com/valdeirpsr/dotfiles/main/install.sh | \
    sudo bash
```

**Forçar a reinstalação do ZSH**

```bash
curl -sSL https://raw.githubusercontent.com/valdeirpsr/dotfiles/main/install.sh | \
    sudo KEEP_ZSHRC="yes" bash
```
